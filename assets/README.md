# Assets

Ce dossier contient les fichiers statiques pour la page de déclarations fiscales.

## Fichier requis

- **Aide-memoire-fiscal-2025.pdf** : Aide-mémoire fiscal complet pour l'année d'imposition 2025

## Instructions

1. Convertir le fichier `Aide-m_moire_fiscal_2025.md` en PDF
2. Renommer en `Aide-memoire-fiscal-2025.pdf` (sans underscore)
3. Déposer ici via GitHub interface ou Git push

Le lien de téléchargement dans `index.html` pointe vers : `assets/Aide-memoire-fiscal-2025.pdf`
